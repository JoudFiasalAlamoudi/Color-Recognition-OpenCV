# Color Recognition using OpenCV

## Project Description
This project detects three colors (Blue, Orange, and Black) using a webcam with OpenCV.

## Tools Used
- Python
- OpenCV
- NumPy
- Anaconda
- Visual Studio Code

## How to Run
1. Create and activate the Anaconda environment.
2. Install the required libraries:
   pip install -r requirements.txt
3. Run:
   python main.py
